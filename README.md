# cocoshell

## Install
You just need `python3` and `pip` installed.
```shell
sudo apt install python3 python3-pip
git clone https://github.com/ConstantinTi/cocoshell.git
cd cocoshell
pip3 install -r requirements.txt
python3 cocoshell.py
```

## Usage
For now, just use `-h` for available arguments and `help` within the program for available commands.
